# Employee Attrition Analysis

## Project Overview

This project analyzes employee attrition data using Python, Pandas, and Matplotlib.

The objective is to identify patterns and factors that contribute to employee attrition through data cleaning, preprocessing, and visualization techniques.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Dataset Features

* Age
* Department
* Gender
* MonthlyIncome
* OverTime
* TotalWorkingYears
* Attrition

## Data Preprocessing

* Handled missing values using mean imputation.
* Removed duplicate records.
* Removed outliers using the IQR method.

## Visualizations

1. Attrition by Department
2. Overtime vs Attrition
3. Employee Age Distribution
4. Experience vs Income
5. Monthly Income Distribution by Gender

## Key Insights

* Analyzed attrition trends across different departments.
* Studied the impact of overtime on employee turnover.
* Examined relationships between work experience and salary.
* Compared salary distributions across genders.
